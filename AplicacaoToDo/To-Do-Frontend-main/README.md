# Download do NodeJS
Baixe o NodeJS em: https://nodejs.org/pt
# Comando para criar projeto no vite
```
npm create vite@latest meu-app -- --template react
```
Se quiser utilizar TypeScript ao invés de JavaScript
```
npm create vite@latest meu-app -- --template react-ts
```
Mais informações em: https://pt.vite.dev/guide/

Utilizar o `npm create` apenas se quiser criar o seu projeto do zero. 
Não é necessário se você clonar esse repositório.
# Comando para instalar todos os plugins listados em package.json
```
npm install
```
# Comando para executar o projeto na sua máquina
```
npm run dev
```
